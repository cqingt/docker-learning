容器与宿主机时间不同步
---


### 解决方式一：追加容器启动参数

```bash
-v /etc/localtime:/etc/localtime:ro

或

-e "TZ=Asia/Shanghai"
```

### 解决方式二：调整 Dockerfile 参数

```bash
ENV TZ=America/Los_Angeles
RUN ln -snf /usr/share/zoneinfo/$TZ /etc/localtime && echo $TZ > /etc/timezone
```
